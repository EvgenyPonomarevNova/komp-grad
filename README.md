# Компьютерная грамотность 🖥️

Платформа для обучения детей 7-12 лет основам работы с компьютером. От полного нуля до уверенного пользователя.

## 🚀 О проекте

Ребенок умеет только играть в Roblox и смотреть YouTube? Наш курс поможет:
- Понять, из чего состоит компьютер
- Научиться работать с файлами и папками
- Освоить операционную систему Windows
- Устанавливать программы без помощи взрослых
- Чувствовать себя уверенно за ПК

## 🛠 Технологии

- **Next.js 14** (App Router)
- **TypeScript**
- **TailwindCSS**
- **shadcn/ui** (с пресетом Maia)
- **Lucide React** (иконки)

## 📦 Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/EvgenyPonomarevNova/komp-grad.git

# Перейти в папку проекта
cd komp-grad

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
